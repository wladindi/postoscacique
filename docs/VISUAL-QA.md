# Checklist visual

Use este checklist em cada pagina antes de marcar a task como concluida.

## Header
- [ ] Logo na mesma proporcao do PNG.
- [ ] Itens de menu com a mesma ordem.
- [ ] Item ativo sublinhado quando existir no PNG.
- [ ] Botao "Contato" com cor, tamanho e alinhamento corretos.

## Tipografia
- [ ] Peso dos titulos confere com o PNG.
- [ ] Quebras de linha principais conferem.
- [ ] Copy exatamente igual ao PNG aprovado.
- [ ] Textos pequenos continuam legiveis no mobile.

## Cores
- [ ] Vermelho Cacique aplicado nos CTAs e destaques.
- [ ] Pretos/cinzas com contraste semelhante ao PNG.
- [ ] Fundos escuros e overlays preservam leitura.

## Imagens
- [ ] Assets originais carregados de `public/assets`.
- [ ] Enquadramento igual ao PNG.
- [ ] Altura e proporcao estaveis entre desktop e mobile.

## Layout
- [ ] Largura de container consistente.
- [ ] Espacamentos verticais conferem.
- [ ] Cards e grids mantem alinhamento.
- [ ] Nenhum texto sobrepoe outro elemento.

## Responsivo
- [ ] Desktop validado em largura de referencia.
- [ ] Tablet validado.
- [ ] Mobile validado.
- [ ] Navegacao mobile funcional.
