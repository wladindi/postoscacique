Coloque aqui os logos, fotos e mockups originais usados nos PNGs.

```text
assets/

  posto-cacique/
```

## Regra de fidelidade

Os componentes ja reservam espaco para imagens, mas os arquivos finais precisam
ser os assets originais para que o site fique exatamente igual aos PNGs.
